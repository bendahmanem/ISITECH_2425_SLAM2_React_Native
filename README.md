# Regles

## Installation

```bash
npx create-expo-app@latest
```

## Lancer le projet

```bash
npx expo start
```

Pour lancer en mode tunnel:

```bash
npx expo start --tunnel
```

## Assets

Telecharger les assets sur Teams et les placer dans le dossier `assets` du projet.

## Reinitialiser le projet

```bash
npm run reset-project
```
